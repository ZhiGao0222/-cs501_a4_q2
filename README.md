# CS501 Assignment 4 - Q2

This project is a simple Android app built with Kotlin and Jetpack Compose.

## What the app does

- asks for location permission  
- shows Google Map  
- moves the map to the user's location  
- adds a marker for the user's location  
- shows address information  
- allows the user to add custom markers by tapping the map  

## How to use

1. Run the app  
2. Tap **Get My Location**  
3. Allow permission  
4. The map will move to the current location  
5. Tap on the map to add markers

- `MainActivity.kt`  
- `LocationMapScreen.kt`  
- `AndroidManifest.xml`
